# Primeiro projeto em ASP.NET

Nesse projeto aprendi a usar o padrão MVC, trabalhar com Entity Framework e aplicar conceitos de OOP. Também aprendi a mapear entidades usando Fluent Mapping e a criar consultas com LINQ, que permite escrever consultas semelhantes a SQL diretamente em C#. Isso é muito interessante, pois permite que o sistema crie e configure o banco de dados automaticamente, tornando-o independente e pronto para uso em qualquer negócio.

Aprendi a implementar autenticação de usuários e gerenciamento de permissões. Usuários administradores têm acesso completo ao sistema, enquanto usuários comuns têm apenas ações limitadas. Para isso, utilizei Claims, JWT e Roles, configurando tudo no arquivo Program.cs.

Também aprendi a usar arquivos estáticos globais para compartilhar informações em todo o sistema. Ao usar com cuidado, isso permite centralizar dados repetitivos e facilita a manutenção do sistema.
